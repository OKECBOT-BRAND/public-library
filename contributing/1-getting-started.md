# Getting Started with Okecbot Development

## Welcome to Okecbot

Okecbot is a complete, local-first ecosystem where users run AI-powered browser agents on their machine, buy or sell automation services in a trusted P2P marketplace, and manage everything through secure, professional tools — all built by DCSSP.

## Our Repositories

| Repository | Purpose | Access |
|------------|---------|--------|
| `okecbot-desktop-app` | Browser AI Agent — runs 1,000 intelligent profiles locally | Private |
| `okecbot-web-api` | Backend core — auth, licensing, escrow, sync | Private |
| `okecbot-marketplace-client` | P2P Marketplace — sell/buy AI workflows & leased sessions | Private |
| `okecbot-web-user-client` | User Dashboard — billing, device control, API keys | Private |
| `okecbot-web-admin-client` | Admin Portal — support, monitoring, dispute resolution | Private |
| `okecbot-web-home` | Brand Site — www.okecbot.com | Private |
| `okecbot-web-blog` | Blog CMS — knowledge, updates, thought leadership | Private |
| `okecbot-compile-assets` | Build tools & binaries for desktop compilation | Private |
| `okecbot-public-library` | Public docs, SDKs, manifestos | Public |

## First Steps

1. **Set up your development environment**
   - Install required tools (Node.js, PHP, Composer, etc.)
   - Set up your IDE with recommended plugins
   - Configure Git with your name and email

2. **Get access**
   - Request access to private repositories from your team lead
   - Set up SSH keys for GitHub authentication

3. **Clone the repository**
   ```bash
   git clone git@github.com:okecbot-org/okecbot-desktop-app.git
   cd okecbot-desktop-app
   ```

4. **Follow the setup guide** in the repository's README.md

5. **Run the application locally** to verify your setup
